# experiments

## Purpose
This repo is for general "vibe coding" projects: web apps, tools, scripts, webpages, etc. And not just final projects, but also as a scratchpad for agents that need a github repo to talk to.

## Structure
Use a subfolder for each project. Work out of that subfolder - you don't need to talk to any of the other subfolders; that will just pollute the context window.
